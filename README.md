# PlugAI

Сайт для [plugai.ru](https://plugai.ru) — экосистема AI-проектов.

## Структура

```
.
├── index.html       # разметка страницы
├── styles.css       # стили (тёмная тема, адаптив)
├── script.js        # интерактив + фоновая анимация
├── assets/
│   └── favicon.svg  # иконка
└── README.md
```

## Локальный просмотр

Просто открой `index.html` в браузере, либо подними статический сервер:

```bash
python3 -m http.server 8080
# или
npx serve
```

Затем зайди на http://localhost:8080

## Деплой

Это статический сайт — разворачивается где угодно:

- **GitHub Pages**: включи Pages для ветки в настройках репозитория
- **Vercel / Netlify**: просто подключи репозиторий
- **Свой сервер**: скопируй файлы в корень веб-хостинга и настрой домен `plugai.ru`

## Что отредактировать под свои проекты

- **Проекты** — в `index.html` секция `#projects`: замени карточки `Проект #1..#4` на свои (название, описание, теги, ссылка, статус — `badge-live` / `badge-beta` / `badge-soon`).
- **Контакты** — почта и Telegram в `#contact` и футере.
- **Мета-теги** — `<title>`, `description`, `og:*` в `<head>`.
- **Цвета** — CSS-переменные в `:root` в `styles.css` (`--accent`, `--accent-2`, `--gradient`).

## Идеи для развития

- Отдельные страницы под каждый проект (`/projects/<slug>.html`)
- Блог / changelog для апдейтов экосистемы
- Единый логин + API (когда проекты объединятся в экосистему)
- Многоязычность (RU/EN)
